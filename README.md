# MAAS Site Manager

MAAS Site Manager is a tool to manage multiple MAAS installations (called 'sites') at the same time.
It centralizes image management for all enrolled sites and provides a statistical overview of all sites and their machines’ statuses, with more features coming soon.

## Installing MAAS Site Manager

The prefered way of deploying MAAS Site Manager is via Charms. See the [MAAS Site Manager Charm](https://charmhub.io/maas-site-manager-k8s) for instructions on how to deploy it.

## Enrolling MAAS Sites

See the [enrollment docs](docs/enrollment.md) for an overview on how to enroll a MAAS Site with MAAS Site Manager.

## Development

To set up a development environment and start contributing to MAAS Site Manager, see the [deployment](docs/deployment.md) and [development](docs/development.md) docs. More information on linting, testing, etc. can be found in [HACKING.md](HACKING.md)

## Bug Reporting

If you find any bugs while using or developing MAAS Site Manager, please report them [here](https://bugs.launchpad.net/maas-site-manager/+filebug). For any security-related issues, see [SECURITY.md](SECURITY.md).
